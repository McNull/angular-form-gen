Form Edit

On this example page existing or new forms can be edited or created.

  • The Form Information tab contains applications specific information fields for the form.
  • The Form Schema tab contains the actual schema editor where form fields can be modified.
  • The Form Preview tab allows the form schema to be previewed and tested. Note that this tab is disabled whenever the current form state is invalid.
  • Click on the (upper right) Save button to store the form.
  • Click on the (upper right) Close button to cancel any modifications.
